# FAQ

## Does this connect to my Shopify store?

No. It only reads a local CSV export. It does not need Shopify admin access.

## Will it import products for me?

No. It creates a cleaned CSV and warning report so you can review before importing.

## What is the paid bundle for?

The paid bundle is for merchants who want import checklists, more examples, and a commercial-use package.

## Refund boundary

Refunds are reasonable when the included examples do not run as described. Store migration, catalog cleanup consulting, and Shopify admin access are not included.
